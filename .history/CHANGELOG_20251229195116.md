- Added applause.wav support: the application will play `applause.wav` after showing salary calculation results.

- Bumped application version to `1.0.2.0` (updated `SalaryCalculator.csproj` and `version.txt`).
- Built single-file self-contained release: `Releases/1.0.2.0/SalaryCalculator.exe` (EXE not committed to git per request).
- Pushed source changes and `applause.wav` to `origin/main`.
